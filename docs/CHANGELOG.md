# Changelog

## Current repository build

- Responsive Android-fit layout with safe-area spacing
- Mobile-friendly scrollable story view
- Archive search, category filters, and pagination
- Persistent cut-up boards per case
- Case-specific state preservation inside each save slot
- Evidence pagination and primary-proof accusation selection
- Local slot export
- Low Hardware, Reduced Motion, and High Contrast settings
- Non-blocking browser audio initialization

## Packaging scope

This repository contains the **web source**. Android packaging should be done through a standard Android Studio / Gradle workflow rather than manually assembling an APK archive.
